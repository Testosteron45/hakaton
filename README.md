# Тревелье

**Тревелье** — мобильное приложение-гид по заведениям Краснодара, которое поможет выбрать идеальное место для отдыха по вашему вкусу. Название объединяет слова *travel* (путешествие) и *сомелье* — как ваш персональный эксперт по заведениям и развлечениям!

## Описание

Тревелье работает как travel-Tinder: пользователь листает карточки с разными местами города — ресторанами, парками, достопримечательностями, кафе. Листайте вправо, если место нравится, влево — если не заинтересовало. На основе ваших лайков приложение предлагает оптимальное место для посещения.

## Основные возможности

- 🔥 Просмотр карточек интересных мест в стиле «свайп влево/вправо»  
- 🏆 Персонализированные рекомендации на основе ваших предпочтений  
- 🗺️ Детальная информация, фото и карта для каждого заведения  
- 💡 Для жителей и гостей города, чтобы находить новые места для отдыха

### Какую проблему решает приложение

Ориентироваться среди множества заведений в большом городе — непросто: рекомендации в интернете могут быть устаревшими, а поисковики предлагают слишком общее или навязчиво рекламируемое. Новичкам сложно понять, куда реально стоит сходить, а местным жителям — найти что-то новое среди “приевшегося” и предсказуемого. Тревелье снимает эти трудности, выступая вашим персональным гидом по городу, учитывающим личные интересы и предлагающим действительно подходящие места для отдыха, развлечений или гастрономических открытий — быстро, удобно и в игровой форме.


## Для кого

- Для туристов, кто впервые в Краснодаре и не знает с чего начать
- Для горожан, которые хотят найти что-то новенькое или оригинальное
- Для всех, кто любит открывать город и отдыхать ярко!


## Как использовать

1. Запустите приложение.
2. Листайте места: нравится — свайп вправо, нет — влево.
3. После серии свайпов получите подборку подходящих для вас мест!
4. Изучайте информацию и отправляйтесь отдыхать.

## Технологии

- Flutter (кросс-платформенная разработка)
- Управление состоянием через Riverpod
- Firebase (авторизация, хранение лайков, т.д.)
- Использование локальной базы (places.json) с описанием всех мест

## Запуск приложения

```
flutter pub get
flutter run
```
