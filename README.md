# Кроссплатформенное приложение по заведениям Краснодарского края

Кроссплатформенное Flutter-приложение для поиска, просмотра и подбора заведений и интересных мест Краснодара и Краснодарского края.

## Описание

Приложение работает как travel-Tinder: пользователь листает карточки с разными местами города и края — ресторанами, парками, достопримечательностями, кафе, ТРЦ и винодельнями. Листайте вправо, если место нравится, влево — если не заинтересовало. После серии свайпов приложение анализирует выбор пользователя и предлагает персональные рекомендации.

## Основные возможности

- 🔥 Просмотр карточек интересных мест в стиле «свайп влево/вправо»  
- 🎯 Тематические режимы сессий: романтика, бюджет, семья, активный отдых и другие  
- 🏆 Персонализированные рекомендации на основе ваших предпочтений  
- 🗺️ Детальная информация, фото и карта для каждого заведения  
- 👤 Профиль пользователя со статистикой и накопленными интересами  
- 🏪 Возможность добавить собственное заведение и смотреть его статистику  
- 💡 Для жителей и гостей города, чтобы находить новые места для отдыха

### Какую проблему решает приложение

Ориентироваться среди множества заведений и локаций в большом городе и регионе — непросто: рекомендации в интернете могут быть устаревшими, а поисковики предлагают слишком общее или навязчиво рекламируемое. Новичкам сложно понять, куда реально стоит сходить, а местным жителям — найти что-то новое среди “приевшегося” и предсказуемого. Приложение помогает решить эту проблему, учитывая личные интересы пользователя и предлагая действительно подходящие места для отдыха, развлечений или гастрономических открытий — быстро, удобно и в игровой форме.

## Как это работает

1. При запуске приложение загружает базу заведений и открывает выбор режима.
2. Пользователь выбирает обычную или тематическую свайп-сессию.
3. Во время сессии показывается 12 карточек мест, а свайпы сохраняются как аналитика.
4. После завершения формируется топ рекомендаций на основе лайков, тегов и пользовательских предпочтений.
5. В профиле можно посмотреть накопленные интересы, а также добавить собственное заведение.

## Как использовать

1. Запустите приложение.
2. Листайте места: нравится — свайп вправо, нет — влево.
3. После серии свайпов получите подборку подходящих для вас мест!
4. Изучайте информацию и отправляйтесь отдыхать.

## Технологии

- Flutter (кросс-платформенная разработка)
- Flutter Web
- Управление состоянием через Riverpod
- Firebase
- Аутентификация — Firebase Auth
- Хранение данных и аналитики — Cloud Firestore
- Навигация — go_router
- Свайп-карточки — appinio_swiper
- Карты и геолокация — flutter_map + latlong2
- Кэширование изображений — cached_network_image
- Локальные ассеты изображений для базовых мест
- Хранение пользовательских настроек — shared_preferences
- 3D-модели и просмотр — model_viewer_plus
- Внешние ссылки — url_launcher
- Встроенные стили Material и Cupertino для нативного UI
